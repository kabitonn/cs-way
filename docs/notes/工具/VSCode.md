### Paste Image 配置

- 文件名前缀：Paste Image: Name Prefix
- 文件存储地址：Paste Image: Path

```
{
    "pasteImage.namePrefix": "mk-",
    "pasteImage.path": "${projectRoot}/assets/cs-note/distribute"
}
```
